# datatrip - A viagem do cliente começa na tela

Este projeto foi desenvolvido como parte do Enterprise Challenge da FIAP em parceria com a ClickBus. O objetivo principal é transformar dados brutos em inteligência estratégica para entender o comportamento de compra e prever ações personalizadas.

## O Desafio
A ClickBus propôs o desafio de aumentar a recorrência de compras através de três frentes:

   Perfil de Compra: Segmentação de clientes (RFM).
   
   Previsão da Próxima Compra: Classificação binária para os próximos 30 dias.
   
   Previsão do Próximo Trecho: Sistema de recomendação de destino (Origem-Destino).


## Tecnologias e Metodologia
Utilizamos uma stack robusta para garantir a escalabilidade e precisão da solução:

  Linguagem: Python (Pandas, Scikit-Learn)
  
  Banco de Dados: Oracle SQL e MongoDB (NoSQL)
  
  Orquestração: Apache Airflow

  Visualização: Power BI
  
  Gestão: Framework Ágil (Scrum) via Azure Boards


### Modelagem Técnica
  Segmentação RFM: Classificamos os clientes em 7 perfis (ex: Melhores Clientes, Em Risco, Adormecidos).
  
  Machine Learning: Modelos de classificação para churn/propensão de compra e algoritmos de recomendação para próximos destinos.
  

## Resultados e Impacto
  Visão 360º: Dashboard interativo para monitoramento de KPIs (Receita Total, Ticket Médio, RFM).
  
  Ações Direcionadas: Identificação de R$ 200M+ em receita proveniente dos melhores clientes, permitindo campanhas de fidelização ultra-     personalizadas.
  
  Previsões Acionáveis: Antecipação das necessidades do cliente, otimizando o ROI de marketing.
